## Description

This repository has a WebSocket server and a sample client to communicate with it. The code for the server program is located in the `src` directory and the code for the sample client is located in the `client` directory.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development server
$ npm run start

# watch mode server
$ npm run start:dev

# production mode server
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
